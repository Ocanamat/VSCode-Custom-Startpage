# Project Docs

This folder contains maintenance, development, and publishing documentation
for the VS Code Custom Start Page extension. Files are plain Markdown for
Obsidian compatibility.

## Contents
- 01-Architecture.md: How the extension works.
- 02-Configuration.md: Settings and user options.
- 03-Webview-Security.md: CSP, iframe sandbox, and navigation limits.
- 04-Development.md: Local dev flow and checklists.
- 05-Publishing.md: Marketplace metadata and publish flow.
- 06-Release-Checklist.md: Step-by-step release checklist.
- 07-Troubleshooting.md: Common errors and fixes.
- 08-Future-Features.md: Ideas and next improvements.
